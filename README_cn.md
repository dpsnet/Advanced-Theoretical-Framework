# 统一场理论高级理论框架

本仓库包含统一场理论的高级理论模型和扩展，专注于M-1至M-9系列的理论模块。

## 仓库结构

### 高级模块
- **M-1**：工具准备与问题提出
- **M-2**：分形时空理论
- **M-3**：自旋与多重扭转理论
- **M-4**：动态拓扑与万花筒效应
- **M-5**：时间原理与动态拓扑基础
- **M-6**：从高维空间到4维时空的谱维映射
- **M-7**：媒介粒子质量与力的长程性的几何推导
- **M-8**：分形高速扭转量子化与多重扭转的微观起源
- **M-9**：高速扭转的存在性证明与基础推导

### 关键概念
- 分形时空结构
- 多重扭转量化
- 谱维流动
- 高维时空映射
- 动态拓扑演化

### 语言版本
本仓库中的所有论文均提供中英文双语版本：
- **中文版**：`_cn.md` 扩展名的文件
- **英文版**：`_EN.md` 或 `_en.md` 扩展名的文件
